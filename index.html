Learning Questionnaire

Answer each statement based on your usual learning habits.

Select a score from 1 to 5 for each statement. Your responses are converted into model features and sent to the FastAPI backend for learning style prediction.

How This Website Works

From questionnaire answers to a learning profile.

1 Answer the questionnaire

Students respond to 15 VAK-based statements using a 1-5 agreement scale.

2 Calculate learning scores

The website calculates Visual, Auditory, and Kinesthetic average scores from the answers.

3 Validate the response pattern

The system checks for unclear or anomalous answers, such as repeated values or conflicting score patterns.

4 Predict and download

Valid scores are sent to the ML model, then the matching learning profile PDF becomes available.
